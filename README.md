This <strong>Support / Resistance indicator</strong> for MT4 brings, besides displaying the <strong>SR price level</strong> for any timeframe chosen by the trade, also the possible <strong>future SR lines</strong>.</div>
<div style="text-align: center;"><h3><a href="#" id="Functions" name="Functions" title="3 Lines Support / Resistance Functions"><strong>3 Lines Support / Resistance</strong> main functions</a></h3></div>
<p>Further more, all this is done <strong>dynamically and automatically</strong>, so when the price reaches and closes a certain <strong>support or resistance line</strong>, this line is <strong>erased as not being valid anymore</strong>, and the <strong>possible SR line becomes the current level</strong>.</p>
